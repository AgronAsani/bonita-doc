1.8.19 Web service connector overview

Generic configuration

  1. Go to Web services in the connectors list.
  2. SelectWeb Service Client.
  3. Click on Next
  4. Enter a name and description for the connector.
  5. Click on Next.

To format your entries according to SOAP specifications, go to SOAP!

See a working example of a web service connector in this tutorial.

Warning: this connector uses the StAX API. StAX libraries cannot be loaded in more than one classloader but must instead be placed in a single location where they can be called by all the items that use them. When you configure a process that uses this connector, you must manage the jar files.
